## Installation

This project uses [uv](https://github.com/astral-sh/uv) for dependency management.

```bash
# Create a venv
uv venv

# Activate the virtual environment
source .venv/bin/activate

# Install GRAIL
uv pip install -e .

# Run the tests
uv run pytest
```
